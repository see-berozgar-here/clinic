## рЯУЭ How to Add Images to Website

### **Method 1 - From URL (Current Method)**
```html
<img src="https://images.unsplash.com/photo-xxx" alt="Doctor">
```
**Pros:** Easy, no file management
**Cons:** Depends on external URL

### **Method 2 - Local File (Recommended)**
```html
<img src="images/doctor.jpg" alt="Doctor">
```
**Pros:** Faster, reliable, professional
**Cons:** Need to organize files

### **Method 3 - Base64 Encoding**
Embed images directly in HTML (not recommended for large images)

## рЯТЊ Image Optimization

### **File Sizes:**
- Hero image: 100-200KB
- Doctor photo: 50-100KB
- Service icons: 10-20KB
- Thumbnails: 10-30KB

### **Optimization Tools:**
- TinyPNG - https://tinypng.com
- Compressor.io - https://compressor.io
- ImageOptim (Mac) - https://imageoptim.com

### **Formats:**
- JPG: Photos (best for quality)
- PNG: Icons/logos (supports transparency)
- WebP: Modern format (smaller size)
- SVG: Vectors (infinitely scalable)

---

## рЯОѓ Image Quality Standards

### **Desktop:**
- DPI: 72 DPI (web standard)
- Format: JPEG for photos, PNG for graphics
- Resolution: 1920x1080 for hero

### **Mobile:**
- Responsive: Scales to screen size
- Format: Same as desktop
- Compression: More aggressive

### **Accessibility:**
- Alt text: Descriptive text for each image
- Lazy loading: Load images as needed
- Color contrast: Sufficient for text overlay
